JSON.parse 与 JSON.stringify(json 格式的使用)

什么是 json 数据

JSON 数据的书写格式是：名称/值对。
名称/值对组合中的名称写在前面（在双引号中），值对写在后面，中间用冒号隔开，
其中 值 可以是：数字（整数或浮点数）、字符串（在双引号中）、布尔值（true 或 false）、数组（在方括号中）、对象（在花括号中）、null
ps：json 数据本质就是字符串

JSON.parse()

从一个字符串中解析出 json 对象

JSON.stringify()
